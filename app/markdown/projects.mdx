### DataForge Studio 🔬
*Full-Stack ML Data Preparation Platform*

[![GitHub](https://img.shields.io/badge/%2F-repo-grey?logo=github&labelColor=%23818181&color=%23008282)](https://github.com/Prajwal18py/SMART-CSV-HEALTH-CHECKER) [![Live Demo](https://img.shields.io/badge/Live-Demo-green?logo=streamlit&labelColor=%23ff4b4b&color=%23008282)](https://dataforge-studio.streamlit.app/)
A production-grade web application enabling **end-to-end ML workflows** — from raw data ingestion to deployable model export.

- 🧩 **13 specialized data processing modules** for complete ML pipelines
- 🤖 AI-powered anomaly detection using **Isolation Forest** + advanced missing data imputation (MICE algorithm)
- ⚖️ Class balancing via **pure NumPy SMOTE/ADASYN** (no external imbalance libraries)
- 🏎️ Automated model training supporting **12 ML algorithms** with real-time data lineage tracking
- 📦 Handles **100,000+ row datasets** with streaming ops; supports CSV, Parquet, JSON, Excel
- 📄 Dark-themed PDF report generation using ReportLab
- 🔧 **Tech Stack**: Python, Streamlit, Pandas, Scikit-learn, Plotly, SQLite, Supabase
